🖱️ 마우스 조작

  • 좌클릭 드래그 (배경): 선택된 도구(✏️ 펜 또는 ↗ 화살표)로 그리기
  • 우클릭 드래그 (배경): 화면 패닝 (지도 스크롤 이동)
  • 마우스 휠: 지도 확대/축소 (마우스 커서 위치 기준)
  • 좌클릭 (챔피언/오브젝트/획): 객체 단독 선택
  • 좌클릭 드래그 (챔피언/오브젝트): 배치된 챔피언 및 스마트 핑 위치 이동
  • 우클릭 (챔피언/오브젝트/선): 컨텍스트 메뉴 오픈 (화살표 추가/삭제 등)

⌨️ 단축키

  • ← / → : 이전 / 다음 스텝 이동
  • Del / Backspace : 선택된 객체 삭제
  • Ctrl + Z : 실행 취소 (Undo)
  • Ctrl + Shift + Z / Ctrl + Y : 다시 실행 (Redo)

📋 스텝 및 라인 관리

  • + Step 추가: 현재 상태를 바탕으로 다음 스텝을 연장하여 애니메이션을 생성합니다.
  • 프리셋 기능: 챔피언을 드래그해서 배치하거나 아군/적군 프리셋 묶음을 전체 드롭하여 배치할 수 있습니다.
